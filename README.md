# IBM HR Employee Attrition Analysis (Tableau)

##  Project Overview
This project focuses on identifying the key factors that lead to employee attrition within an organization using the **IBM HR Analytics dataset**. The analysis aims to provide actionable insights for HR departments to improve employee retention strategies.

##  Tools Used
* **Visualization Tool:** Tableau Desktop
* **Data Source:** IBM HR Analytics Dataset (Kaggle)
* **Techniques:** Exploratory Data Analysis (EDA), KPI Tracking, and Trend Analysis.

## Key Insights from Analysis
1. **Departmental Impact:** The Research & Development department shows the highest volume of attrition cases.
2. **Overtime Factor:** Employees working overtime have a significantly higher attrition rate, indicating a link between burnout and turnover.
3. **Age Risk Group:** The analysis reveals that employees in the **25-35** age bracket are most prone to leaving the organization.
4. **Job Roles:** Roles such as Laboratory Technicians and Sales Executives experience higher turnover compared to managerial positions.
